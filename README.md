# How to Install:
First go your Ternimal. Then

Step1: git clone https://github.com/Vill4inS/Hunter_Janowar.git

Step2: cd Hunter_Janowar

Step3: cdmod +x install.sh

Step4: ./install.sh

Step5: chmod +x janowar.sh

Your installation has been completed
